# variableLabel

## Dependencies
 - Node/Npm (NVM)
 	- base-64
 	- js-crc
 	- sha256

### Setup

	git clone git@github.com:mathew-fleisch/variableLabel.git
	cd variableLabel
	npm install

### Updating

	git pull origin master
	rm -rf node_modules # (optional)
	npm install

### Usage

	node index.js

	# or 

	node base64.js
